# RUGIPP REGISTRI - FLASK APP

## Kreiramo **venv**
    python -m venv venv
    
## Aktiviramo **venv** 
    venv\Scripts\activate

## Instalacija requirements
    pip install -r requirements.txt

## Aktiviramo FLASK_APP
    set FLASK_APP=app.py
    
## Aktiviramo mod za greske FLASK_DEBUG
    set FLASK_DEBUG=1

## Pokrenemo aplikaciju 
    flask run
